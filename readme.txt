
Pour pouvoir lancer le script:
- Installer Python: https://www.python.org/downloads/
- Configurer Python sous Windows: 
    - Aller dans Système
    - Paramètres système avancés
    - Variables d'environnement...
    - Dans Variables utilisateur et/ou dans Variables système, ajouter le chemin d'accès de Python et des scripts de base, du style:
        - C:\Program Files (x86)\Python38-32\
        - C:\Program Files (x86)\Python38-32\Scripts\

Il est ensuite possible de lancer la commande python depuis n'importe où en console.

Il faut également installer la librairie pyautogui (sert à effectuer les clicks):
> python -m pip install pyautogui

Pour lancer le script:
> python programme.py

Normalement c'est tout.
